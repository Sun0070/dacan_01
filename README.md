# DAKER Platform
### 긴급 인수인계 해커톤: 명세서만 보고 구현하라

데이터 경진대회(해커톤) 정보 탐색, 팀 구성, 제출, 랭킹 확인을 한 곳에서 할 수 있는 **풀스택 없는 React 웹 플랫폼**입니다.
백엔드 없이 **localStorage만으로** 모든 상태를 관리하며, 최초 실행 시 시드 데이터를 자동으로 생성합니다.

## 기술 스택

| 항목 | 내용 |
|---|---|
| 프레임워크 | React 18 + TypeScript |
| 번들러 | Vite 5 |
| 라우터 | React Router v6 |
| 스타일링 | Tailwind CSS v3 (`darkMode: 'class'`) |
| 데이터 저장 | `localStorage` (백엔드 없음) |
| 폰트 | Pretendard Variable (CDN) |
| 아이콘 | 커스텀 SVG 아이콘 시스템 (30개+) |

## 실행 방법

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm run dev
# → http://localhost:5173 에서 확인

# 3. 프로덕션 빌드
npm run build

# 4. 빌드 결과 로컬 미리보기
npm run preview
```

> 별도 설정 없이 실행하면 앱 최초 실행 시 8개 해커톤, 18개 팀, 6개 리더보드 데이터가 자동으로 생성됩니다.

## 페이지별 기능

---

### 공통 — Navbar & 글로벌 검색

모든 페이지 상단에 고정되는 네비게이션 바입니다.

**Navbar**

| 요소 | 동작 |
|---|---|
| `[DA]` 로고 | 클릭 시 홈(`/`)으로 이동 |
| 네비게이션 링크 | 해커톤 / 팀 찾기 / 랭킹 / 나의 공간, 현재 페이지 인디고 하단 보더 표시 |
| 검색 버튼 | 클릭 또는 `Ctrl+K` (`Cmd+K`) 단축키로 검색 모달 열기 |
| 다크 모드 버튼 | Sun/Moon 아이콘 토글, 설정 localStorage 유지 |
| 모바일 햄버거 | 768px 이하에서 드롭다운 메뉴 표시 |

**글로벌 검색 모달 (`Ctrl+K`)**

- `Ctrl+K` / `Cmd+K` 또는 검색 버튼으로 열기, `Esc` / 배경 클릭으로 닫기
- 입력과 동시에 실시간 결과 반영
- **검색 대상**: 해커톤 제목·태그, 팀 이름·소개
- **결과 그룹화**: 해커톤 / 팀 섹션으로 분리 표시, 항목 클릭 시 해당 페이지로 이동

---

### 홈 (`/`)

서비스 진입점 페이지입니다.

**히어로 섹션**
- 그리드 패턴 배경 + 방사형 그라디언트 오버레이
- 인디고-바이올렛 그라디언트 메인 헤드라인
- `해커톤 탐색` → `/hackathons`, `팀 찾기` → `/camp` 버튼
- 진행 중인 해커톤이 있을 때 실시간 펄스 뱃지 표시

**기능 카드 (3개)**
- 해커톤 / 팀 찾기 / 랭킹 카드에 실시간 수치 표시
  - 대회 수 / 모집 중인 팀 수 / 총 제출 수
- 호버 시 카드 부상 + 그림자 효과

**통계 행**
- 전체 해커톤 수 / 모집 중인 팀 수 / 총 제출 수를 3분할 표시

**이용 방법 (3단계)**
- ① 해커톤 탐색 → ② 팀 구성 → ③ 제출 & 랭킹
- 각 단계 사이 커넥터 라인으로 흐름 시각화

**주요 해커톤 섹션**
- 진행중 → 예정 → 종료 순으로 최대 3개 카드 표시

### 해커톤 목록 (`/hackathons`)

**레이아웃**: 좌측 사이드바 + 우측 카드 그리드

#### 좌측 사이드바 (데스크탑 고정, 스티키)

| 필터 | 동작 |
|---|---|
| 검색 입력 | 제목·태그 실시간 필터링, X 버튼으로 초기화 |
| 상태 필터 | 전체 / 진행중 / 예정 / 종료 — 각 상태 카운트 뱃지, 진행중 항목 펄스 |
| 정렬 | 최신순(종료일 기준) / 마감임박순(제출 마감일 기준) |
| 태그 필터 | 전체 태그를 칩으로 표시, 클릭 시 필터 적용 |
| 초기화 버튼 | 활성 필터가 있을 때만 노출 |

#### 우측 카드 패널

- 결과 수 표시 (필터 적용 시 "N개 결과" / 전체 시 "N개의 해커톤")
- 반응형 그리드: 모바일 1열 → 태블릿 2열 → 데스크탑 3열
- 결과 없을 시 빈 상태 안내 + 필터 초기화 버튼

#### HackathonCard 컴포넌트

- 상태별 3px 상단 컬러 보더: 진행중 `emerald` / 예정 `violet` / 종료 `slate`
- D-day 뱃지: D-3 이하 `rose` / D-14 이하 `amber` / 그 외 `slate`
- 우측 상단 북마크 버튼 (localStorage 저장)
- 호버 시 `-translate-y-1` 부상 + 그림자 효과

### 해커톤 상세 (`/hackathons/:slug`)

**상단 헤드 영역**
- 상태 뱃지 + D-day 뱃지
- 해커톤 제목, 태그 목록
- 제출 마감일 / 종료일
- 북마크 버튼 / 외부 링크 버튼

**8탭 스티키 탭 바** — 스크롤 시 상단 고정

| 탭 | 주요 내용 |
|:---:|---|
| 개요 | 대회 소개 요약, 팀 구성 정책 (솔로 허용 여부, 최대 인원) |
| 안내 | 공지사항 목록, 규칙/FAQ 외부 링크 버튼 |
| 평가 | 평가 메트릭 설명, 일일 최대 제출 횟수, 런타임 제한, 점수 구성 비율 |
| 일정 | 마일스톤 타임라인 — 과거(체크)/현재(펄스)/미래(빈원) 시각적 구분 |
| 상금 | 1~3위 메달 아이콘 + 상금 금액 카드 |
| 팀 | 해당 해커톤 등록 팀 목록, 팀 찾기 페이지 이동 링크 |
| 제출 | 허용 아티팩트 유형, 제출 가이드, 제출 폼 |
| 리더보드 | 순위표 (팀명 / 점수 / 제출 시각), 점수 구성 세부 표시 |

**제출 폼**
- 팀 이름, 플랜 제목, 웹 URL, PDF URL 입력
- 제출 시 localStorage 저장 + 성공 토스트 알림

### 팀 찾기 (`/camp`)

**레이아웃**: 좌측 사이드바 + 우측 팀 카드 그리드

#### 좌측 사이드바 (데스크탑 고정, 스티키)

| 필터 | 동작 |
|---|---|
| 검색 입력 | 팀 이름·소개 실시간 필터링 |
| 모집 상태 필터 | 전체 / 모집중 / 마감 — 카운트 뱃지 + 색상 도트 |
| 해커톤 필터 | 진행중/예정/종료 그룹별 목록, 각 해커톤의 등록 팀 수 뱃지 |
| 역할 필터 | Frontend / Backend / Designer / ML Engineer / Data Scientist / PM |
| 초기화 버튼 | 활성 필터가 있을 때만 노출 |

#### 우측 팀 카드 패널

- **내 스킬 기반 추천 팀** (나의 공간에서 스킬 설정 시 상단 노출)
  - 내 스킬과 매칭되는 모집중 팀을 인디고 배경 섹션으로 강조 표시
- 결과 수 표시
- 반응형 그리드: 모바일 1열 → 태블릿 2열 → 데스크탑 3열

#### 팀 만들기 / 수정 모달

- **기본 정보**: 팀 이름(필수), 해커톤 선택(필수), 팀 소개
- **모집 설정**: 모집 여부 토글 스위치, 현재 인원 스텝퍼(1~10), 찾는 역할 다중 선택
- **연락처**: 오픈채팅 등 URL 입력
- 모달 외부 클릭 / 취소 버튼으로 닫기

#### TeamCard 컴포넌트

- 팀 코드 해시 기반 결정적 색상 아바타 (6가지 색상)
- 모집 상태 뱃지: 모집중 `emerald` / 마감 `slate`
- 멤버 수 도트 인디케이터 (최대 5dot 표시)
- 내가 만든 팀에만 수정/삭제 버튼 노출
- 삭제 시 확인 다이얼로그 표시

### 랭킹 (`/rankings`)

**레이아웃**: 좌측 사이드바 + 우측 리더보드 패널

#### 좌측 사이드바 (데스크탑 고정, 스티키) — 3단 구성

| 섹션 | 내용 |
|---|---|
| 검색창 | 해커톤 이름 실시간 필터링, 항상 최상단 고정 |
| 내 참여 대회 | 내가 팀을 만든 해커톤 목록, 현재 순위 뱃지 표시, 검색 시 자동 숨김 |
| 대회 목록 | 진행중 / 예정 / 종료 그룹별 전체 해커톤 목록, 각 항목에 참여 팀 수 뱃지 |

- 각 섹션은 Camp·Hackathons 사이드바와 동일한 디자인 패턴 적용 (섹션 헤더 uppercase, 항목 라운드 버튼, 활성 항목 인디고 배경)
- 앱 로드 시 첫 번째 진행중 해커톤 자동 선택
- 사이드바 내부 독립 스크롤 지원

#### 우측 리더보드 패널

- 선택된 해커톤 제목 + 마지막 업데이트 시각
- **TOP 3 포디움**: 1~3위 팀을 메달 아이콘(금/은/동)과 높이 차이로 시각화
- **순위표**: 팀명, 점수, 제출 시각, 참가자·심사위원 세부 점수
- **내 팀 강조**: 내가 만든 팀의 행은 인디고 배경 + "내 팀" 뱃지로 구분
- 아티팩트 버튼: 웹 사이트 링크 / PDF 보고서 링크
- 리더보드가 없는 해커톤 선택 시 빈 상태 안내

**모바일 대응**: 상단 드롭다운 셀렉터로 해커톤 선택

### 나의 공간 (`/me`)

- 이름 입력

- 스킬 다중 선택: Python / JavaScript / TypeScript / React / Node.js / Machine Learning / Deep Learning / SQL / Docker / Figma
- 커스텀 스킬 직접 입력 추가
- 저장 즉시 팀 찾기 추천 기능에 반영

#### 빠른 이동 네비게이션

북마크한 해커톤 / 내 팀 / 제출 이력 섹션으로 스크롤 이동 버튼

#### 북마크한 해커톤 섹션

- 북마크한 해커톤을 카드 그리드로 표시

- 팀 없을 시 팀 만들기 링크 안내

#### 제출 이력 섹션

- 내가 제출한 모든 내역을 타임라인으로 표시
- 해커톤명 / 팀명 / 제출 시각
- 플랜 제목 / 웹 URL / PDF URL 아티팩트 링크 버튼

## 확장 기능

### 다크 모드

- Navbar의 Sun/Moon 아이콘 버튼으로 즉시 전환
- Tailwind CSS `darkMode: 'class'` 방식, `document.documentElement` 에 `dark` 클래스 토글
- `daker_theme` 키로 localStorage에 저장, 새로고침 후에도 유지
- 모든 페이지 / 컴포넌트 / 모달에 다크 스타일 완전 대응

### 스켈레톤 로딩

- 데이터 로드 전 shimmer 애니메이션 스켈레톤 표시
- 해커톤 카드형 / 팀 카드형 두 가지 variant 지원

### 토스트 알림

- 화면 우하단에 스택 형태로 쌓임
- 3초 후 자동 소멸
- 타입별 색상 구분: `success` (에메랄드) / `error` (로즈) / `info` (슬레이트)
- 사용 시점: 팀 생성/수정/삭제, 제출 완료, 프로필 저장

### 해커톤 타임라인

- 해커톤 상세 → 일정 탭에서 마일스톤 시각화
- 상태 구분: 과거(회색 체크) / 현재(인디고 펄스 애니메이션) / 미래(빈 원)

### 스킬 기반 팀 추천

- 나의 공간에서 스킬 설정 시 팀 찾기 페이지 상단에 추천 섹션 자동 노출
- 내 스킬과 `lookingFor`가 겹치는 모집중 팀을 인디고 배경으로 강조

### 북마크 시스템

- 해커톤 카드 북마크 아이콘으로 저장/해제
- `UserProfile.bookmarks` 배열에 slug 저장
- 홈 / 해커톤 목록 / 해커톤 상세 / 나의 공간 전 페이지에서 상태 동기화

### 랭킹 내 팀 인식

- `UserProfile.myTeamCodes` 기반으로 내가 만든 팀을 리더보드에서 자동 식별
- 랭킹 사이드바 **"내 참여 대회"** 섹션: 내 팀이 있는 해커톤만 최상단에 모아 표시, 현재 순위 뱃지 함께 노출
- 리더보드 테이블에서 내 팀 행은 인디고 배경 하이라이트 + **"내 팀"** 뱃지로 시각적 구분

### 통일된 사이드바 레이아웃

해커톤 목록 / 팀 찾기 / 랭킹 세 페이지가 동일한 사이드바 디자인 패턴을 공유합니다.

- 섹션 헤더: uppercase 추적 텍스트
- 항목: 라운드 버튼, 우측 카운트 뱃지
- 활성 항목: 인디고 배경 + 인디고 텍스트 + 인디고 뱃지
- 데스크탑 스티키 + 내부 독립 스크롤, 모바일 토글 패널

### 반응형 레이아웃

| 화면 크기 | 레이아웃 |
|---|---|
| 모바일 (< 768px) | 햄버거 메뉴, 필터 토글 버튼으로 사이드바 열기 |
| 태블릿 (768px~1024px) | 2컬럼 카드 그리드 |
| 데스크탑 (> 1024px) | 스티키 사이드바 + 카드 그리드 전체 레이아웃 |
